• 支援 @username 和完整網址格式
• 每行一個連結，保持清晰排版
• 一般網址 (https://) 也會自動轉換為連結

地址設定

💡 個人地址建議：
• 可填入工作地址、居住地區或留空
• 0 / 50 字（建議不超過 50 字）
• 避免過於詳細的個人住址以保護隱私
⚠️ 超過建議字數可能影響 NFC 容量
💡 GPS 座標說明：
• 使用十進位格式 (例如: 25.033964, 121.564468)
• 可從 Google Maps 取得座標: 右鍵點擊地點 → 點擊座標即可複製
• 座標會自動轉換為 Google Maps 連結，點擊即可導航
• 留空則不顯示地址資訊

個人化設定

💡 請使用完整的圖片網址：
• Imgur - 上傳後右鍵複製圖片網址
• PostImages - 上傳後複製 Direct link
• GitHub - 上傳到 repository 後使用 raw 連結
• Google Drive - 分享連結會自動轉換為直接連結
• 任何支援 HTTPS 的圖床服務
⚠️ 必須以 https:// 開頭的完整網址，避免使用相對路徑
📋 Google Drive 使用方式：上傳圖片 → 右鍵分享 → 複製連結即可
💡 個人版面問候語建議：
• 0 / 60 字（建議不超過 60 字）
• 簡潔有力的自我介紹
• 可使用 emoji 增加親和力
⚠️ 超過建議字數可能影響 NFC 容量
